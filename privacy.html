Privacy

BlocNav is designed for use in environments where trust, safety, and data integrity matter.

This public website does not collect personal data beyond standard web server logs. We do not run analytics trackers, advertising pixels, or behavioral profiling.

BlocNav’s application platform is currently in private pilot evaluation. Data handling practices within the application depend on deployment context and are governed by pilot agreements.

As a design principle: local data ownership, minimal collection, and explicit operational control are prioritized.

This policy may evolve as BlocNav matures. Updates will be reflected here.
